Swoole-Cli

介紹

Swoole-Cli 是一個 PHP 的 binary 集成發行版，集成了 swoole, php kernel, php-cli, php-fpm 等等多個常用拓展。 Swoole-Cli 是全靜態編譯打包而成，不需要依賴任何的作業系統 so 動態鏈結庫；具有良好的移植性， 可以在任意 Linux 系統間複製。

# 顯示版本
swoole-cli -v

# 顯示當前模組
swoole-cli -m

# 軟連結 -> php
ln -s /usr/bin/swoole-cli /usr/bin/php

配置文件

swoole-cli 內建已經有集成 php-fpm 所以可以使用 -P 來啟動 fpm

參數

  • -h 幫助

  • -P 啟動 php-fpm

    • -p 啟動 php-fpm 的進程管理器前綴名稱
    • -F, --nodaemonize
    • -R, --allow-to-run-as-root

  • -c 指定 php.ini 使用 （可傳入 路徑 或是 檔案，傳入路徑會去尋找路徑下的 php.ini

  • -d 自定義 INI 的值 (-d foo=bar)

# 查看幫助
swoole-cli -P -h

# 執行 fpm 
swoole-cli -P --fpm-config /opt/php-8.1/etc/php-fpm.conf -p /opt/php-8.1/var

# 使用 root 執行
swoole-cli -P --fpm-config /opt/php-8.1/etc/php-fpm.conf -p /opt/php-8.1/var -F -R

安裝 Swoole

1. 自行編譯 swoole-src

  • 編譯相關需求:
    • 4.8 -> PHP 7.2 以上
    • 5.0 -> PHP 8.0 以上
    • gcc-4.8 以上
    • make
    • autoconf
    • phpize

# 已經發行的版本
# https://github.com/swoole/swoole-src/releases

cd swoole-src

phpize
./configure
make
sudo make install

# 編譯完後在 php.ini 中新增拓展
extension=swoole.so

2. PECL 安裝

1

pecl install swoole
